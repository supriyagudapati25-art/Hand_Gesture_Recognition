# Hand_Gesture_Recognition
A Hand Gesture Recognition System built using Convolutional Neural Networks (CNN), TensorFlow, and OpenCV. This project classifies 10 different hand gestures from images using the LeapGestRecog dataset as part of the SkillCraft Technology Machine Learning Internship.
# Hand Gesture Recognition using CNN

## SkillCraft Technology - Machine Learning Internship

### Task 04: Hand Gesture Recognition

## Project Overview

This project develops a Convolutional Neural Network (CNN) capable of recognizing and classifying different hand gestures from images. The model is trained on the LeapGestRecog dataset and demonstrates high classification accuracy.

## Features

- Hand Gesture Classification
- Image Preprocessing using OpenCV
- CNN Model using TensorFlow/Keras
- Model Evaluation
- Model Saving for Future Predictions

## Technologies Used

- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib
- Scikit-learn

## Dataset

LeapGestRecog Dataset (Kaggle)

## Model Performance

- Training Accuracy: 99.68%
- Validation Accuracy: 99.98%
- Test Accuracy: 99.975%
